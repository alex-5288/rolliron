# Icon assets

Drop these files in here before deploying:

- `icon-192.png` — 192×192 PNG, transparent or black background
- `icon-512.png` — 512×512 PNG, transparent or black background
- `apple-touch-icon.png` — 180×180 PNG for iOS home screen
- `favicon.ico` (optional)

**Quick make-it-yourself path:**

1. Sketch a black square with a neon-red "R&I" or barbell+gi icon in any image editor.
2. Export at 192 / 512 / 180. Save here with the exact names above.
3. Commit + redeploy.

Until these exist, iOS will show a screenshot of the page as the home-screen icon (still works — just ugly).
